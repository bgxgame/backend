## 项目概述

这是一个名为"My-Life-Planner"（我的生活计划器）的Rust后端项目，基于Axum框架开发，用于管理个人计划和任务。

## 技术栈

- **Web框架**: Axum (Rust)
- **异步运行时**: Tokio
- **数据库**: PostgreSQL (通过SQLx)
- **数据序列化**: Serde/serde_json
- **密码哈希**: Argon2
- **JWT认证**: jsonwebtoken
- **日志追踪**: tracing

## 核心功能

### 1. 用户系统
- **用户注册**: 创建新用户账号，密码使用Argon2加密存储
- **用户登录**: 验证用户身份并生成JWT令牌

### 2. 计划管理
- **创建计划**: 用户可以创建新的计划/任务
- **获取计划**: 可以获取公开的计划列表
- **更新计划**: 修改已有的计划信息
- **删除计划**: 删除指定的计划

### 3. 权限控制
- **公开/私有**: 计划可以设置为公开或私有
- **身份验证**: 受保护的API端点需要有效的JWT令牌

## 数据模型

### 用户表 (users)
- ID (主键)
- 用户名 (唯一)
- 密码哈希
- 创建时间

### 计划表 (plans)
- ID (主键)
- 标题
- 描述
- 状态 (待办、进行中、已完成、归档)
- 分类 (工作、生活、学习等)
- 优先级
- 截止日期
- 公开性 (公开/私有)
- 创建和更新时间

## API端点

- `GET /api/plans` - 获取公开计划列表
- `POST /api/register` - 用户注册
- `POST /api/login` - 用户登录
- `POST /api/plans` - 创建新计划（需认证）
- `PATCH /api/plans/:id` - 更新计划（需认证）
- `DELETE /api/plans/:id` - 删除计划（需认证）

## 项目特点

1. **安全性**: 使用Argon2进行密码哈希，JWT进行身份验证
2. **CORS支持**: 支持跨域请求，便于前端集成
3. **错误处理**: 完善的错误处理机制
4. **数据库迁移**: 包含数据库结构的迁移脚本
5. **环境配置**: 通过.env文件管理配置

这是一个功能完整的计划管理后端系统，具备用户认证和计划CRUD功能，适合与前端应用配合使用。
